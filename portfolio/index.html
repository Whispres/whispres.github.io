I have a diverse range of samples to showcase my capabilities and growth as a technical writer. The following page hosts an overview of technical documentation and exerpts from recent technical writing projects, as well as samples I have developed specifically for my portfolio.

Quick Reference Phlebotomy Guide

  • Software: Microsoft Word 2016
  • Featured Skills: Information Design, Chemistry Knowledge, Health Sciences

    • This phlebotomy guide is a visual aid for nurses and lab technicians. It provides quick access to blood collection and specimen transport information, but is not an exhaustive list of all sampling protocols. Users are expected to be fully trained and the guide acts as a quick reference or reminder. The hospital name is fictional and information has been altered to protect the content owner's privacy.

    Standard Operating Procedure Template

  • Software: Microsoft Word 2016
  • Featured Skills: SOP writing, scientific writing, good lab practices, analytical statistics

    • This template provides a general discussion of developing a nonclinical laboratory SOP. Examples and suggestions are included for implementing statistical information, procedure data, health and safety concerns, regulatory requirements and best style practices.

    API Endpoint URL Map

  • Software: Microsoft Visio 2016 and Postman
  • Featured Skills: RESTful API, CURL and Diagram Design

    • API endpoint URL maps are designed to help users quickly navigate and consume API endpoints. During API development, the map provides an overview of the API architecture and visualizes basic resource information. This map features a list of the parameters available for the resources and their associated methods. API information included in this map was derived from CURL testing and Postman.
